# trmnl-maps

Pre-rendered public-domain historical map crops (grayscale, tonal levels baked in) for a
personal TRMNL e-ink plugin ("I Fucking Love Maps"). Source: David Rumsey Map Collection
(IIIF). Served via the jsDelivr CDN so the e-ink renderer loads them instantly.

Regenerate with `prerender_pool.py` in the workspace.
